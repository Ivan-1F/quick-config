## quick-config

这是一个库模组来帮助你快速构建 `malilib` 风格的配置界面

这是一个客户端 mod，不依赖 Fabric API

![preview](https://github.com/Ivan-1F/quick-config/blob/fabric-1.15.2/screenshots/preview.png?raw=true)
